Corral's Vector Calculus

Here you can download Corral's Vector Calculus. It is a modified version of Vector Calculus written by Michael Corral and released under GNU Free Documentation License. I was making changes while teaching a calculus course at Penn State. I do not plan to continue, unless I will have to teach the same class again.

I did not plan to publish this book myself, but was not able to contact Michael Corral. Please let me know if you have a way to contact him.

The source could be obtained from https://github.com/anton-petrunin/calc3book; if you install Git, the following command will create a local copy of the source code for you.

git clone https://github.com/anton-petrunin/birkhoff.git
