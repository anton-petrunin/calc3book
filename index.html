Corral's Vector Calculus

Tis site is the homepage for a modified vesion of "Vector Calculus" by Michael Corral. The modifications were made by Anton Petrunin, while teaching a calculus course at Penn State, Fall 2016. Both, the modified version and the original are written under GNU Free Documentation License. (Michael Corral made a pledge to keep the book's contents the same modulo corrections of errors and typos.)

You can download the latest version of our "Corral's Vector Calculus" here: calc3book.pdf.

Hopefully this page will become a home for a community developed book. I invite everyone to participate, "Wikis" and "Issues" on GitHub should make it easy; you could also write me an email (which is easy to find).

The source could be obtained from https://github.com/anton-petrunin/calc3book; if you install Git, the following command will create a local copy of the source code for you.

git clone https://github.com/anton-petrunin/calc3book.git

Write me a short note if you use the book.
