Corral's Vector Calculus

This is the homepage for a modified version of "Vector Calculus" by Michael Corral. The modifications were made by Anton Petrunin while teaching a calculus course at Penn State, Fall 2016. Both, the modified version and the original are written under the GNU Free Documentation License. (Michael Corral has pledged to keep the book's contents the same, except for corrections of errors and typos.)

You can download the latest version of "Corral's Vector Calculus" here: calc3book.pdf.

W hope this page will evolve into a hub for a community-developed book. We invite everyone to participate; "Wikis" and "Issues" on GitHub should make it easy. You can also reach out via email (which is easy to find).

The source code can be obtained from https://github.com/anton-petrunin/calc3book.

Write me a short note if you use the book.
